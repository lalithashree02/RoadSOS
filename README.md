Road SOS - Emergency Assistance Web App

Road SOS is a responsive, full-stack web application designed to assist drivers during roadside emergencies. The application provides an interactive live tracking dashboard map that locates nearby essential emergency services like hospitals, police stations, puncture shops, and towing services based on geographical coordinates.

Features

* Dual-Interface Routing: Includes a clean marketing landing page that smoothly transitions into the core application dashboard.
* Interactive Live Mapping: Visualizes your current location and maps out critical nearby emergency utilities.
* Emergency Datasets: Powered by comprehensive localized datasets for rapid lookup of critical services.
* Direct Emergency Calling: Integrated one-touch dialing functions (e.g., immediate 112 calling triggers) for mobile devices.

Tech Stack

Backend: Python, Flask, Flask-CORS
Data Processing: Pandas, Math (Haversine formula calculations for distance)
Frontend: HTML5, CSS3, JavaScript (Geolocation APIs & Interactive Mapping)
